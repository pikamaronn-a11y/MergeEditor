# MergeEditor

Monaco Editor（VS Codeと同じエンジン）をベースにした、マージ機能付きWindowsデスクトップエディタです。  
外部エディタでファイルが更新された際、**更新時点で自動的に差分を検出**し、インラインで**未保存箇所との衝突解決**を行うことができます。

## 特徴

- **マージ機能**: ファイルを外部エディタで更新した際、自動的に差分を検出してインラインで衝突解決パネルを表示。ブロックごとに「自分を保持」か「外部を取り込む」を選択できます。
- **ファイル監視**: 開いているファイルの外部変更をリアルタイムで検知し、ハイライト表示で変更箇所を通知します。
- **ミニマップ連動**: マージ検出箇所がミニマップ上にオレンジ色で表示されます。
- **マルチウィンドウ**: 複数のウィンドウを同時に開いて独立して操作可能です。
- **多彩な書式**: Plain Text, JavaScript, TypeScript, HTML, CSS, JSON, Python, Markdown に対応。
- **3種類のテーマ**: Light, Dark, High Contrast から選択可能。
- **右端折り返し**: 画面端での折り返し表示をON/OFF可能（ステータスバーに状態表示）。

## 必要環境

- Windows 10 以降
- Node.js 18 以上（開発時のみ）

## インストール

### ビルド済みパッケージからインストール
`dist/MergeEditor Setup 1.0.0.exe` を実行してインストールしてください。

### ソースからビルド
```bash

# 依存パッケージをインストール
npm install

# アプリを起動（開発モード）
npm start

# インストーラーをビルド
npm run build
```

ビルド完了後、`dist/MergeEditor Setup 1.0.0.exe` が生成されます。

### ポータブル版をビルド
インストール不要で実行できるポータブル版をビルドする場合：
```bash
npm run build:portable
```

ビルド完了後、`dist/MergeEditor 1.0.0.exe` が生成されます。このファイルは単体で実行可能です。

## 使い方

### 基本操作
| 操作 | ショートカット |
|---|---|
| 新規ウィンドウ | `Ctrl+N` |
| ファイルを開く | `Ctrl+O` |
| 上書き保存 | `Ctrl+S` |
| 名前を付けて保存 | `Ctrl+Shift+S` |
| ウィンドウを閉じる | `Alt+F4` |

### 表示メニュー
| 操作 | ショートカット |
|---|---|
| 折り返しトグル | `Ctrl+Shift+W` |
| ミニマップトグル | `Ctrl+Shift+M` |
| ズームイン | `Ctrl+=` |
| ズームアウト | `Ctrl+-` |
| ズームリセット | `Ctrl+0` |
| 全画面表示 | `F11` |

### マージ操作
1. ファイルを開くと、自動的にファイルの変更監視が開始されます。
2. 外部エディタでファイルを更新すると、差分が検出され、衝突箇所にインラインパネルが表示されます。
3. パネル内で「✓ 自分を保持」または「↗ 外部を取り込む」を選択して解決します。
4. すべてのブロックを一括で解決する場合は、画面下部のツールバーを使用します。
