# 2026/6/23 更新 v1.2.0

## 追加
- 表示メニューに「文字コード」サブメニューを追加しました。UTF-8 / UTF-16 LE / UTF-16 BE / Shift_JIS / EUC-JP を選択できます。
- ステータスバーに現在の文字コードを表示し、クリックで切り替えられるようにしました。
- 文字コードを変更すると、開いているファイルを選択した文字コードで再読み込みします（未保存の変更がある場合は内容を保持し、保存時の文字コードのみ変更します）。
- 未保存の状態でウィンドウを閉じようとした場合、保存確認のダイアログが表示されるようにしました。
- 対象のテキストファイルパスを指定して起動できるようにしました。ファイルのアプリ関連付けが可能になりました。

## 修正
- エディタにテキストがある状態でファイルを開くと別のウィンドウで開くようにしました。
- 衝突解決用UIの表示を調整しました。また、UI用フォントの情報が反映されるようにしました。

# 2026/6/21 更新 v1.1.0

## 追加
- エディター部のフォント設定機能を追加しました。
- UI部のフォント設定機能を追加しました。
- ステータスバーの各要素をクリックすることで折り返しのONOFF変更、フォント設定ダイアログ起動、行列情報をクリップボードにコピーする機能を追加しました。
- Ctrl＋マウスホイール上下でフォントサイズを変更できるようにしました。

## 修正
- ズームイン、ズームアウトでシステムUIまで影響を受ける事象を修正しました。
- ステータスバーのフォントサイズを調整しました。
- レンダリングエンジンのズーム率が何らかの原因で100%以外となっている場合に調整するように修正しました。
- 保存ダイアログを開いた際、ファイルの種類に選択中の書式が反映されるように修正しました。
